자동 백업 프로그램 사용 안내

프로그램 이름: 자동 백업 프로그램 (ModernBackupApp)
버전: 1.0
개발 환경: Python 3.15
작성자: 이은수

1. 프로그램 개요
- 선택한 원본 폴더를 지정한 저장 폴더로 자동으로 백업하는 GUI 프로그램입니다.
- 백업 주기를 사용자가 설정할 수 있으며, 백업 진행 상황과 로그를 실시간으로 확인 가능합니다.
- Windows 환경에서 실행 가능하며, 작업 표시줄 아이콘이 프로그램용으로 표시됩니다.

2 사용법
프로그램 창이 열리면 다음 설정을 진행합니다:
   - 원본 폴더(Source): 백업할 폴더 선택
   - 저장 폴더(Target): 백업을 저장할 폴더 선택
   - 백업 주기: 30초, 1분, 5분 등 원하는 주기 선택

3. 사용법
1) 모든 폴더를 선택하고 주기를 설정한 후, "자동 백업 시작" 버튼을 클릭합니다.
2) 실행 중에는 설정 변경이 불가능하며, 사이드바 입력창이 비활성화됩니다.
3) 타이머가 표시되며, 주기마다 자동으로 백업이 실행됩니다.
4) 실시간 작업 로그에서 백업 상태를 확인할 수 있습니다.
5) 자동 백업 중지:
   - "백업 중지 (Stop)" 버튼 클릭 → 자동 백업 종료
   - 설정 변경 가능

5. 주의사항
- 원본 폴더와 저장 폴더는 반드시 존재해야 합니다.
- 백업 폴더 안에는 날짜와 시간이 붙은 폴더가 생성됩니다.
- 대용량 폴더를 백업할 경우, 시간과 디스크 용량을 확인하세요.
- Windows 환경에서만 아이콘 설정이 적용됩니다.

6. 로그 예시
[12:00:01] 원본 경로 설정됨: C:\Users\User\Documents
[12:00:05] 저장 경로 설정됨: D:\Backup
[12:00:10] 자동 백업 시작됨 (주기: 5분)
[12:05:10] ✅ 백업 완료: Documents_20260111_120510
